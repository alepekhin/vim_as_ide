## Install ollama

- sudo apt install curl

- ollama run carstenuhlig/omnicoder-2-9b:latest

## Install git and clone projects

## Configure nvim

- sudo apt install ripgrep
- add nerd fonts

wget -P ~/.local/share/fonts https://github.com/ryanoasis/nerd-fonts/releases/download/v3.0.2/JetBrainsMono.zip \
&& cd ~/.local/share/fonts \
&& unzip JetBrainsMono.zip \
&& rm JetBrainsMono.zip \
&& fc-cache -fv

After installing, open your terminal’s settings/preferences and set the Font to JetBrainsMono Nerd Font


- copy vim_as_ide/config/nvim tp ~/.config
